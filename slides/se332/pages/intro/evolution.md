# Part 2: Evolution (Simple Cache → Multi-Model)

## Timeline

| Year | Milestone | Impact |
|------|-----------|--------|
| **2009–2015** | Basic Cache & Key-Value Store | Foundation |
| **2015–2019** | Persistence, Replication, Cluster | Enterprise-ready |
| **2020+** | Redis Modules ecosystem | Multi-model expansion |
| **2024** | License shift (SSPL/RSALv2) | Legal restructure |

## The Transformation

<div class="mt-6">

**From:** Extracting data for computation elsewhere

**To:** Bringing logic closer to the data itself

</div>

<div class="mt-4 p-4 bg-blue-50 rounded border border-blue-200">

**Multi-Model Era:** Document DB (JSON), Vector DB (AI/RAG), Graph, TimeSeries all in one platform

</div>
