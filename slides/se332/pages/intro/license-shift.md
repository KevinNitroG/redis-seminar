# Part 3: The 2024 License Shift

## The Event

Transitioned from **open-source BSD** to **SSPL/RSALv2** (starting with Redis 7.4)

## Root Cause

Preventing major cloud providers (AWS, Google, Azure) from monetizing the software without contributing back

## New Rules

- ✅ Still free for **internal/SaaS use**
- ❌ **Bans** offering competitive Managed Services/DBaaS without commercial license
- Impact: Cloud providers can't offer cheap Redis as a service

## Community Reaction

<div class="mt-4 space-y-2">

**Valkey** — True open-source fork backed by:
- Linux Foundation
- AWS, Google
- Community support

**Enterprise Decision:** Reassess legal/compliance risks

</div>
