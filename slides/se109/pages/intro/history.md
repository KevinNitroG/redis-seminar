# A Brief History

- **2009** — Salvatore Sanfilippo (antirez) created Redis for real-time analytics at his startup
- **2010** — VMware sponsored development; Redis gained persistence, replication
- **2013** — Redis Cluster (horizontal scaling) introduced
- **2015** — Redis Labs founded; Redis Sentinel for high availability
- **2020** — Redis Modules ecosystem: RedisJSON, RediSearch, RedisGraph, RedisTimeSeries
- **2024** — License change (see next slide)
